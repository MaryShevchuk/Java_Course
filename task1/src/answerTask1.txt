Типы данных в Java.
1) На лекции мы познакомились со следующие типы данных:byte, short, char, int, long, float, double, boolean.
Чем отличаются эти типы?

На прошлом уроке мы познакомились с примитивными типами данных.
В Java существует 4 группы примитивных типов данных (всего получается 8 примитивных типов данных).
Для каждого из 8 типов строго определен набор допустимых значений и разрешенных операций.
8 типов данных отличаются диапазоном значений, которые может хранить переменная, а также функциональностью использования.

Далее рассмотрим 4 группы примитивных типов данных:

1. Тип целых чисел:
- byte - 8 разрядное число (8 бит: -128 ... 127) - удобен при работе с необработанными данными, который поступает из файла или сетевого подключения (+экономит в ассивах память)
- short - 16 разрядное число (16 бит: -32768 ... 32767)
- int - 32 разрядное число (32 бита: -2147483648 ... 2147483647) - удобен при работе с целочисленными данными (если нет проблем с паматью)
- long - 64 битное число (64 бита: -9223372036854775808 ... 9223372036854775807) - удобен для работы с большими целыми числами

2. Тип чисел вещественных :
- float - 32 разрядное число (32 бита: -3.4E+38 ... 3.4E+38) - удобен для использования, когда не требуется особой точности в исчислении дробной части числа.
- double - 64 битное число (64 бита: -1.7E+308 ... 1.7E+308) - также  удобен для использования, когда не требуется особой точности в исчислении дробной части числа.

3. Тип логических данных:
- boolean - хранит значение true или false - удобен для опрделения того, является ли условие истинным.

4. Тип символьный:
- char - это беззнаковое целое число, представляющее собой символы UTF-16 (буквы и цифры) - удобен для хранения любого символа.

2) Какие типы Вы знаете ещё?
Помимо примитивного типа данных, существует тип данных ссылочный.
К ссылочным типам данных (переменные) относятся все типы данных, которые создаются с помощью конструкторов классов.
Они предназначены для доступа к объектам

String  предназначен для хранения строк текста.
